## 📋 What is Nova-Client
Nova-Client acts as a bridge for your internet connection. It improves your browsing experience by routing data through secure servers. This tool combines a user-friendly interface with an automatic scanner. The scanner identifies the best connection paths for your specific location. You do not need technical skills to use these features. Nova-Client manages the complex network tasks in the background while you focus on your work.

## ⚙️ Initial setup
Launch the application by clicking the new icon on your desktop. The first window shows a status dashboard. You need a configuration link or file from your network provider to start the service.

1. Copy your configuration details provided by your service operator.
2. Open the "Subscriptions" or "Servers" tab inside Nova-Client.
3. Paste the link into the provided field and click "Import".
4. The client automatically populates the list of server options.
5. Select a server that displays low latency numbers to ensure the fastest speed.
6. Click the large toggle button in the center of the window to activate the connection.

## 🛡️ Using the built-in scanner
The software includes a tool called Nova Radar. This tool tests different server connections to identify which ones perform best. Use this feature if your internet connection feels slow or unstable.

1. Navigate to the "Radar" or "Scan" tab.
2. Click the "Start Scan" button.
3. Wait for the progress indicator to complete as the tool tests various network nodes.
4. Review the results list. Blue markers usually indicate stable, fast servers.
5. Click "Apply" to update your active connection settings with the best result found by the scan.

## 🛠️ Troubleshooting common issues
If you experience trouble, follow these simple solutions:

* Connection fails: Verify your internet connection works without the app. If other websites fail, restart your router. 
* Application does not open: Ensure you installed the latest version from the link provided above. Some older Windows versions require an update to the .NET framework.
* Slow speeds: Run the scanner again. Sometimes server congestion occurs during peak hours. Switching to a different location in the list often fixes this immediately.
* Cannot import links: Re-copy your link from the source. Ensure you select the full address without adding spaces or extra characters.
